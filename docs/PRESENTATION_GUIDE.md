# Presentation Guide - Mahob Taste

## Slide 1 - Title

- Project: Mahob Taste
- Topic: Restaurant website
- Tools: HTML, Tailwind CLI, JavaScript

## Slide 2 - Assignment requirements

- Minimum 5 pages
- Tailwind CLI
- Dark/Light mode
- GitHub + GitHub Pages
- Responsive design

## Slide 3 - Site map

- Home (`index.html`)
- Menu listing (`menu.html`)
- Dish detail (`detail.html`)
- Authentication (`auth.html`)
- About (`about.html`)
- Contact (`contact.html`)

## Slide 4 - Design inspiration

- Reference: mahobkhmer.com
- Key influence:
  - Warm Khmer color atmosphere
  - Elegant typography
  - Food-focused storytelling sections

## Slide 5 - Technical workflow

- Tailwind source file: `assets/css/input.css`
- Generated CSS: `assets/css/output.css`
- Build command: `npm run build`
- Shared behavior: `assets/js/main.js`

## Slide 6 - Features demo

- Responsive navbar with mobile toggle
- Dark/light mode with saved preference
- Listing cards + detail page
- Contact and login/register pages

## Slide 7 - Quality and structure

- Beginner-friendly folder layout
- Reusable classes and shared scripts
- Easy to maintain and extend

## Slide 8 - Conclusion and next steps

- Requirements completed
- Ready for GitHub Pages deployment
- Future upgrades:
  - Backend authentication
  - Reservation system
  - Dynamic menu filtering
